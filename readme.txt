How to Launch City Bus Manager Text


Make sure Python 3.7+ is installed:
python --version
or
python3 --version


Open a terminal or command prompt and go to the game folder:
cd path/to/CBMText


Run the game:

Windows:
python game.py

macOS / Linux:
python3 game.py

android:
1. Download Termux from F-droid - https://f-droid.org/F-Droid.apk
2. Allow storage access with 'termux-setup-storage'
3. Go to the game folder with 'cd storage/shared/Download/CBMText_V1.1.2'
4. run the gamw with 'python3 game.py'

Tip: If “python not found” appears on Windows, try:
py game.py
